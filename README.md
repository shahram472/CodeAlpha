# Cybersecurity Internship Tasks

This repository contains the tasks completed as part of my cybersecurity internship submission. The projects focus on understanding common cybersecurity threats and defensive mechanisms.

## 📌 Tasks Included

### 1. Phishing Attack Awareness (Presentation)

This task focuses on understanding phishing attacks and how users can identify and prevent them.

**Contents:**

* PowerPoint presentation explaining phishing attacks
* Types of phishing attacks
* Real-world examples
* Prevention techniques and best practices

**Objective:**
To spread awareness about phishing attacks and educate users on how to recognize and avoid them.

---

### 2. Network Intrusion Detection System (NIDS)

This task focuses on studying how network intrusion detection systems work and how they help detect malicious activities within a network.

**Contents:**

* Presentation explaining NIDS
* Overview of network security threats
* How intrusion detection systems monitor network traffic
* Common IDS techniques (signature-based and anomaly-based detection)

**Objective:**
To understand how network monitoring tools help detect unauthorized access or malicious activity in a network.
